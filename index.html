HOW TO PLAY

MOVE: WASD / ARROW KEYS
LOOK: MOUSE MOVEMENT
INTERACT/GRAB: SPACE BAR
USE BOMB: B KEY
SHOOT (PISTOL): F KEY
CROUCH / STAND: C KEY (hold)
JUMP: X KEY
TOGGLE MAP: M KEY
THROW GRENADE: T KEY
PLACE BLOCK: V KEY
DESTROY BLOCK: E KEY
CYCLE BLOCK TYPE: G KEY
Tip: Walk up to NPCs and animals to talk to them! Math problems give Gold. Don't touch the enemies!
